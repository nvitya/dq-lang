# What is the DQ?

DQ is an universal, human-friendly programming language.

**Universal**, because it intended to cover many use cases including:

* Scripts / tools
* Server applications
* Embedded applications
* GUI applications
* Games

**Human friendly** because DQ syntax and library interfaces are designed to make code easy to read and easy to write by humans.

DQ uses well-established programming concepts, like object inheritance with virtual functions.

The DQ compiler is implemented in C++ using LLVM.

The DQ runtime library is implemented in DQ.
